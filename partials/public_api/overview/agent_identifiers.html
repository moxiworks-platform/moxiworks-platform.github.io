Agent Identifiers

The MoxiWorks Platform provides two different primary keys for Agents to allow maximum flexibility for differing use-cases.

If you have already existing Agent data, agent accounts and your own user interface that Agents can use to integrate your account with their MoxiWorks Platform account then you should use moxi_works_agent_id. This identifier guaranteed to be unique, but may be either an alphanumeric string or an email address. It is intended for use cases where integration is managed by end-user interaction.

If you are pulling Agent data from The MoxiWorks Platform and integrating with your own system in a managed or automated fashion, then using agent_uuid is preferable. agent_uuid is guaranteed to be unique, uniformly formatted and to never change for the lifetime of the account.
